01

三种格式

.MHTML最还原。Chrome 原生捕获——完整保留页面状态和所有资源。mhtml
.SINGLE自包含。单个 .html,所有资源内联为 data URL。html
.ZIP可携带。index.html + 文本 + 消息 + 资源,拿来就能交给代理。zip
02

为代理而生

每份归档都附带一层面向 agent 的可读文本:代码块保留缩进,对话页还会解析成结构化的回合。真实样本如下:

text-content.txt

1 加 1 等于几?

等于 2。

def add(a, b):
    return a + b

无损提取 · 代码完整

messages.json

[
  { "role": "user",
    "text": "1 加 1 等于几?" },
  { "role": "assistant",
    "text": "等于 2。" }
]

按 DOM 结构切分回合 · 支持 ChatGPT 与 Claude

03

安装

⬇ 下载 .zip 或从源码克隆 ↗
  1. 01下载 .zip 并解压。
  2. 02打开 chrome://extensions。
  3. 03打开右上角的开发者模式开关。
  4. 04点 加载已解压的扩展程序，选择 pagesnap/ 文件夹。

然后打开任意页面,点 PageSnap 图标,选格式保存。详见 README。
