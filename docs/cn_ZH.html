PageSnap

把任意网页保存成忠实、AI 可读的归档。

在 GitHub 上查看 安装

为什么用 PageSnap

Chrome 自带的保存会丢懒加载内容、还把可读文本搅乱。PageSnap 把一切捕获下来,既给人看,也给代理用。

📦

三种格式

MHTML、自包含的 Single HTML、或可携带的 Web Archive ZIP。按需选择保真度。

🤖

为 AI 而生

不破坏性纯文本(代码块完整),对话页还会输出结构化的 messages.json 回合。

🧩

无损捕获

先滚动懒加载内容,再快照完整渲染后的 DOM,不漏关键内容。

🍃

轻量开源

原生 JS,无构建步骤,无运行时依赖。MIT 协议。

安装

作为解压扩展加载到 Chrome。

git clone https://github.com/jinhuang712/page-snap.git
  1. 打开 chrome://extensions。
  2. 开启 开发者模式。
  3. 点 加载已解压的扩展程序,选择克隆下来的 page-snap 文件夹。

然后打开任意页面,点 PageSnap 图标,选格式保存。详见 README。
