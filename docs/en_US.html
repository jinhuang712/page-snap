PageSnap

Save any web page as a faithful, AI-readable archive.

View on GitHub Install

Why PageSnap

Chrome's built-in save loses lazy-loaded content and mangles readable text. PageSnap captures everything and packages it for both humans and agents.

📦

Three formats

MHTML, a self-contained Single HTML, or a portable Web Archive ZIP. Pick the fidelity you need.

🤖

Built for AI

Non-destructive plain text with code blocks intact, plus structured conversation turns in messages.json.

🧩

Lossless capture

Scrolls lazy-loaded content first, then snapshots the full rendered DOM. Nothing important is left behind.

🍃

Tiny & open

Vanilla JS, no build step, no runtime deps. MIT licensed.

Install

Load it as an unpacked extension in Chrome.

git clone https://github.com/jinhuang712/page-snap.git
  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Click Load unpacked and choose the cloned page-snap folder.

Then open any page, click the PageSnap icon, pick a format, and save. See the README for details.
